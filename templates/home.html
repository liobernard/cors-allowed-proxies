Need a list of proxies for web crawling or other purposes? sslproxies.org is a great resource, but its web server does not have CORS headers configured to accept cross-origin requests. This shouldn't be a problem in general, but when attempting a GET request via your web browser (e.g. clicking a button to get proxies) you'll likely be hit with an error informing you that your browser has blocked the request.

So here's an alternative! A neat proxy server that will make the request for your browser. Cross origin GET requests are allowed here, so just make a request to the super simple API explained below. The server retrieves the payload from sslproxies.org without issue, then parses to a useful format, and finally gets them back to you.


Usage

/proxies/     Returns an array of <= 100 elite/highly anonymous SSL proxies
/proxies/csv/     Returns an CSV-formatted file of <= 100 elite/highly anonymous SSL proxies
/proxy/     Returns one (1) elite/highly anonymous SSL proxy, tested for workability at httpbin.org/ip (may take a few seconds)
